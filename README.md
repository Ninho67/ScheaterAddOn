# Scheater

Votre assistant tout-en-un pour la régulation de vos chauffages électriques.

## Fonctionnalités
- **Régulation**
  - Intégrez vos chauffages dans home assistant et laissez Scheater s'occuper de leur contrôle
- **Planification** : 
  - Créez vos programmes de chauffe par pièce(s) en fonction de vos présences
  - Réduisez votre consommation en cas d'absence
- **Suivi des consommations** : 
  - Configurez votre tarification (heures pleines/creuses)
  - Suivez en temps réel votre consommation d'énergie et optimisez vos factures
